## 技巧

* 按下之后不松手，到达跳跃最高点之后再马上第二次跳跃如此往复。



## Idea

* 死亡越少，越厉害
* 时间越少，越厉害



## 玩家跳跃能力的等级

* 跳跃，单方向。
* 能够达到二级跳的高度，加上左右移动。
* 三段跳的高度，最高高度。
